1. Descrierea proiectului

Perfume Magazine (PerMa) este o aplicatie web care reprezinta un magazin virtual. Acesta pune la dispozitie utilizatorilor/cumparatorilor mai multe parfumuri.

2. Introducere

Proiectul este un instrument Web de online shopping, care permite vizualizarea si achizitionare de parfumuri, grupate pe sezoane.

4. Funcționalitate

  1. Aplicația rulează la 127.0.0.1:5000. Utilizatorul își poate alege sezonul pentru care vrea sa vada lista parfumurilor. În funcție de alegerea făcuta, va fi redirectionat spre pagina corespunzatoare.
  2. De asemenea, utilizatorul poate adauga un produs in cos, poate vizualiza raportul paginii web și pagina de contact.
  3. Panoul de administrare rulează la 127.0.0.1:5000/admin-statistics.html. Un administrator poate genera statistici referitoare la stocuri si comenzi, in formatele CSV, HTML si PDF

5. Arhitectura si procesul de construire al aplicatiei

  1. A fost abordata o arhitectură bazată pe (micro)servicii.
  2. Codul sursă este structurat conform arhitecturii MVC.
  3. Backendul aplicației a fost realizat în Node.js.
  4. Am folosit GitHub ca sistem de stocare şi management online al codului-sursă.

5.1 Backend - API & Admin Panel

Pentru realizarea CRUD API-ului, am recurs la Sequelizer, un ORM (Object Relation Model) care mapeaza tabelele existente in baza de date(perfumes, clients, commands) pe modelele descrise in /src/api/entities/ . Am folosit paradigma MCV (Model, View, Controller) pentru proiectarea API-ului. Aplicatia cuprinde si un admin panel, care poate fi accesat la: 127.0.0.1:5000/admin-statistics.html, care permite generarea si descarcarea rapoartelor privind stocuri si comenzi in formatele HTML, CSV si PDF

5.2 Frontend and design

Pentru realizarea designului aplicatiei, am folosit HTML5, CSS si Javascript. Designul este simplu si potrivit pentru functionalitatea aplicatiei.
